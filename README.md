## What is QuestForge?

QuestForge turns dry **System Design interview questions** into Pixar-style animated fantasy quests.

You pick a challenge (or type your own). The app generates:
- ⚔️ A **quest title** in the style of a game level
- 📜 A **fantasy story** that maps system design components to fantasy characters
- 💡 A **real architectural lesson** hidden inside the story
- 🎨 A **Pixar-style cartoon scene** showing the system design visually — unique every time

Every quest teaches something real about distributed systems — but feels like watching an animated movie.

## How the AI Works

### Groq + Llama 3.1 — Quest Generation

QuestForge uses the **Groq API** with **Llama 3.1 8B Instant** for all quest narrative generation.

```
POST https://api.groq.com/openai/v1/chat/completions
Authorization: Bearer gsk_...
```

**Why Groq?**

| | |
|---|---|
| ✅ **Genuinely free** | 14,400 requests/day, no credit card, no billing ever |
| ✅ **Works everywhere** | No region restrictions — unlike Google Gemini which blocks the free tier in many countries |
| ✅ **Lightning fast** | Groq's custom LPU chip — responses in under 1 second |
| ✅ **Reliable JSON** | `response_format: json_object` forces clean structured output every time |

The prompt maps system design concepts to fantasy characters:

```
WebSocket Server   →  The Crystal Ball of Telepathic Tethers
Load Balancer      →  The Gatekeeper / Toll Bridge Guardian
Database           →  Royal Scroll Vaults (ACID-sealed)
Cache              →  Pouch of Instant Recall
Message Queue      →  The Raven Courier Guild
Rate Limiter       →  The Hourglass Sentry
CDN                →  The Network of Magic Mirrors
Microservice       →  Summoned Golems / Mage Guilds
```

Groq returns a JSON object with `title`, `story`, `lesson`, and `image_scene`. If the primary model hits a rate limit, the app automatically falls back to `gemma2-9b-it` and `llama3-8b-8192`.

### Pollinations.ai — Pixar-Style Image Generation

The quest illustration is generated by [Pollinations.ai](https://pollinations.ai) — completely free, no API key needed.

```js
const url = `https://image.pollinations.ai/prompt/${encodedPrompt}?width=832&height=448&model=flux&nologo=true`;
```

The image prompt is built from `image_scene` — a character-action description that Groq generates alongside the story, describing exactly which fantasy characters appear and what they are visibly doing. This makes every image directly represent the system design concept.

## The Journey: Claude → Gemini → Groq

QuestForge **started with the Anthropic Claude API** (`claude-sonnet-4-6`). When the credit balance ran out:
```
Error 400: Your credit balance is too low to access the Claude API.
```

Switched to **Google Gemini** (advertised as free), but hit a harder wall — the free tier quota was literally `0` for this key due to Google Cloud region restrictions:
```
Error 429: Quota exceeded — free_tier limit: 0, model: gemini-2.0-flash
```

Switched to **Groq** — free for everyone, no exceptions, faster than both. Each migration was a one-file change: same prompt, same JSON parsing, different endpoint.

## Local Setup

```bash
git clone https://github.com/mayageva11/questforge.git
cd questforge
open index.html   # or just drag into your browser
```

1. Get a free Groq key at **[console.groq.com](https://console.groq.com)** — 30 seconds, no credit card
2. Click ⚙️ → paste your `gsk_...` key → Save
3. Pick a challenge → **Forge Quest 🔮**

## Deploy Your Own

1. Fork this repo
2. **Settings → Pages → Deploy from branch → main → / (root)**
3. Wait ~2 min → live at `https://YOUR_USERNAME.github.io/questforge`

No config needed. No environment variables. Users bring their own free Groq key.

## How It Was Deployed

The entire project — repo creation, GitHub Pages setup, every commit — was done through **Claude Code** (Anthropic's AI coding assistant) without ever opening the GitHub web UI.

```bash
# Create repo via GitHub REST API
curl -X POST https://api.github.com/user/repos \
  -H "Authorization: token $TOKEN" \
  -d '{"name":"questforge","private":false}'

# Enable GitHub Pages
curl -X POST https://api.github.com/repos/mayageva11/questforge/pages \
  -H "Authorization: token $TOKEN" \
  -d '{"source":{"branch":"main","path":"/"}}'
```
