## What is this?
QuestForge turns Playwright and automation architecture interview questions into retro RPG quests — with fantasy narrative, pixel-art SVG scenes, and a real architectural lesson in every quest. Powered by Claude AI.

## Why I built this
While preparing for Senior QA Automation Engineer interviews, I kept hitting the same wall: automation System Design questions felt abstract and intimidating. Instead of grinding another doc, I built QuestForge to translate every challenge into a retro RPG quest — because learning sticks better when it doesn't feel like studying. Built during my job search. Shipped because why not.

## How it works
1. 🧙 Select a Quick Quest or type your own automation challenge
2. ⚔️ Claude AI transforms it into a fantasy quest with narrative + pixel-art scene
3. 💡 Get the real architectural lesson hidden in the story
4. 🗺️ Copy to LinkedIn and flex on the timeline

## Tech stack
- Vanilla HTML/CSS/JS — zero dependencies, zero build step
- Claude API (claude-sonnet-4-6) called directly from browser
- SVG rendered inline from AI-generated code
- Deployed via GitHub Pages

## Quick Quests included
- Design a Playwright framework for a monorepo
- Handle flaky tests at scale
- CI/CD pipeline with parallel test execution
- Test data management across environments
- Visual regression testing system
- Page Object Model for 500+ tests
- Cross-browser test strategy
- API + UI test layers

## Local setup
1. Clone the repo
2. Open index.html in browser
3. Enter your Claude API key (stays local only)
4. Start forging quests ⚔️

## Deploy your own
Fork → Settings → Pages → Deploy from main → done.
